# Animated Background

A cosmic starry background animation for websites. This project creates an immersive, visually appealing animated starfield that can be easily integrated into any web project.

## ✨ Features

- 🌌 Beautiful cosmic starry animation
- 💫 Smooth, performant animations
- 🎨 Customizable appearance
- 📱 Responsive design
- ⚡ Lightweight and easy to integrate
- 🔧 Simple to use and configure

## 📋 Tech Stack

- **JavaScript** (76.9%) - Core animation logic
- **HTML** (13.7%) - Structure
- **CSS** (9.4%) - Styling and animation effects

## 🚀 Getting Started

### Prerequisites

No dependencies required! This is vanilla JavaScript.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Nirzarah/animated-background.git
cd animated-background
```

2. Include the files in your project or run locally:
```bash
# Open index.html in your browser
open index.html
```

### Usage

Simply include the animated background in your HTML:

```html
<!-- Include the necessary files -->
<link rel="stylesheet" href="style.css">
<script src="script.js"></script>
```

The animation will automatically start when the page loads.

## 🎯 How It Works

This project creates a cosmic starry background using:
- Canvas API or DOM elements for rendering stars
- JavaScript for animation logic and star generation
- CSS for smooth transitions and effects

## 🛠️ Customization

You can customize various aspects of the animation:
- Number of stars
- Star size and brightness
- Animation speed
- Colors and gradients
- Parallax effects

Check the source code comments for customization options.
